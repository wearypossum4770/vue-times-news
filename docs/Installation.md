## Runtime
- NodeJS
  - HomeBrew (Brew) Installer (Automatically setup)
  ```sh 
    brew install nodejs 
  ```
  - Standalone installer (Manual Setup)
  ```sh 
    https://nodejs.org/en/download/
  ```
- Bun
  - Standalone Install (Auto Setup)
  ```sh
  curl https://bun.sh/install | bash
  ```
- Deno 
  - HomeBrew (Brew) Installer (Automatically setup)
  ```sh
    brew install deno 
  ``` 
  - Standalone Install (Auto Setup)
  ```sh
    curl -fsSL https://deno.land/install.sh | sh
  ```
